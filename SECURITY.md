# Security Policy

## Supported Versions
This project is currently developed on the `master` branch. Security fixes are applied to the latest version.

## Reporting a Vulnerability
If you believe you have found a security vulnerability:

1. **Do not** open a public GitHub issue.
2. Provide a minimal report including:
   - affected version / commit
   - reproduction steps
   - impact assessment
   - any suggested fix

If a private reporting channel is not available in your fork, please open a **draft** pull request with a failing test that reproduces the issue, without exploit details.

## Scope
Sentinel Stream is intended for defensive monitoring and log processing. It should not be used to harm systems.
